# 投诉取证匹配工具 - 使用说明

## 🛠️ 三个独立工具

### 工具一：取证预览工具（Web 界面）

**功能**：解压压缩包 + 预览图片 + 勾选需要识别的截图

**安装依赖**：
```bash
pip install streamlit pillow pandas openpyxl
```

**使用方法**：
```bash
streamlit run evidence_viewer.py
```

**操作步骤**：
1. 浏览器会自动打开 Web 界面（通常是 http://localhost:8501）
2. 点击左侧"上传取证压缩包"按钮
3. 选择你的 .zip 文件
4. 等待解压完成
5. 浏览所有图片，勾选需要识别的截图
6. 点击"导出选中图片列表"，下载 `selected_images.csv`

---

### 工具二：OCR 识别工具

**功能**：批量识别截图 → 提取单号+日期 → 导出 Excel

**使用方法**：
```bash
python ocr_extractor.py --input selected_images.csv --output complaint_records.xlsx
```

**输出**：
- `complaint_records.xlsx` - 包含列：单号、日期、来源图片、来源压缩包

---

### 工具三：Excel 匹配工具

**功能**：匹配总表和 OCR 结果 → 写入【取证来源】列

**使用方法**：
```bash
python excel_matcher.py \
  --master 总表.xlsx \
  --ocr complaint_records.xlsx \
  --output 总表_已匹配.xlsx
```

**参数说明**：
- `--master`: 你的投诉总表 Excel
- `--ocr`: OCR 识别结果（工具二导出的）
- `--output`: 输出文件名（默认：总表_已匹配.xlsx）
- `--id-col`: 总表中投诉单号列名（默认：投诉单号）
- `--date-col`: 总表中投诉日期列名（默认：投诉日期）

**输出**：
- `总表_已匹配.xlsx` - 总表 + 新增【取证来源】列
- `未匹配列表.xlsx` - 未能匹配的记录

## 📋 完整工作流程

```
1️⃣ 上传压缩包
   ↓
2️⃣ Web 界面勾选截图
   ↓
3️⃣ 导出 CSV
   ↓
4️⃣ OCR 识别
   ↓
5️⃣ 导出 Excel（单号+日期+来源）
   ↓
6️⃣ 匹配总表
   ↓
7️⃣ 导出结果（总表_已匹配.xlsx）
```

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install streamlit pillow pandas openpyxl pytesseract

# 2. 启动 Web 工具
streamlit run evidence_viewer.py

# 3. 在浏览器中操作，导出 CSV 后：

# 4. OCR 识别
python ocr_extractor.py --input selected_images.csv --output complaint_records.xlsx

# 5. 匹配总表
python excel_matcher.py --master 你的总表.xlsx --ocr complaint_records.xlsx
```

---

## ⚠️ 注意事项

1. **Tesseract OCR** 需要先安装系统级依赖：
   - CentOS/RHEL: `yum install -y tesseract tesseract-langpack-chi_sim`
   - Ubuntu/Debian: `apt install -y tesseract-ocr tesseract-ocr-chi-sim`

2. **列名匹配**：工具三的 `--id-col` 和 `--date-col` 要和你的总表列名一致

3. **日期格式**：确保总表中的日期格式为 `YYYY-MM-DD`（如 2026-02-24）

4. **内存占用**：如果图片数量巨大（>1000 张），可以分批处理

## 🆘 常见问题

**Q: Web 界面打不开？**
A: 检查防火墙，或使用 `streamlit run evidence_viewer.py --server.port 8502` 换个端口

**Q: OCR 识别不准确？**
A: 可能是图片质量不佳，可以尝试：
- 提高截图分辨率
- 调整图片对比度
- 使用更高精度的 OCR 引擎（如 PaddleOCR）

**Q: 匹配率很低？**
A: 检查：
- 总表中的日期格式是否统一
- 单号是否有前导零或空格
- 日期列是否包含时间（需要先清理）
