---

## 🚀 快速开始

### 第一步：安装 Python

1. 访问：https://www.python.org/downloads/
2. 下载 **Python 3.7 或更高版本**
3. 安装时**勾选 "Add Python to PATH"**
4. 安装完成后**重启电脑**

### 第二步：一键安装

1. 双击运行 **`一键安装.bat`**
2. 脚本会自动：
   - 检查 Python 版本
   - 安装 Python 依赖包
   - 检查 Tesseract OCR

3. 如果提示 Tesseract 未安装：
   - 下载：https://github.com/UB-Mannheim/tesseract/wiki
   - 下载文件：`tesseract-ocr-w64-setup-5.4.0.exe`
   - 双击运行安装程序
   - **重要**：安装时勾选 **Chinese (Simplified)** 语言包
   - 默认安装路径：`C:\Program Files\Tesseract-OCR`
   - 安装完成后**重启电脑**
   - 再次运行 **`一键安装.bat`** 检查

### 第三步：启动工具

1. 双击运行 **`启动工具.bat`**
2. 浏览器会自动打开：http://localhost:8501
3. 开始使用！

---

## 🛠️ 手动安装（如果自动安装失败）

### 安装 Python 依赖

打开命令行（CMD），执行：

```bash
pip install streamlit pillow pandas openpyxl pytesseract tqdm
```

或使用清华镜像（国内更快）：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 安装 Tesseract OCR

1. 下载：https://github.com/UB-Mannheim/tesseract/wiki
2. 下载 `tesseract-ocr-w64-setup-5.4.0.exe`
3. 双击运行安装程序
4. **勾选 "Chinese (Simplified)" 语言包**
5. 安装到：`C:\Program Files\Tesseract-OCR`
6. 设置环境变量：
   - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 新建系统变量：
     - 变量名：`TESSDATA_PREFIX`
     - 变量值：`C:\Program Files\Tesseract-OCR\tessdata`
   - 在 Path 变量中添加：
     - `C:\Program Files\Tesseract-OCR`
7. **重启电脑**

## 📋 使用教程

### 标签页 1：取证预览

1. 上传压缩包（.zip 格式）
2. 自动解压并显示所有图片
3. 点击按钮选择需要的图片（✓ = 已选）
4. 点击「导出选中图片」

### 标签页 2：OCR 识别

1. 自动读取选中的图片
2. 点击「开始 OCR 识别」
3. 等待识别完成（进度条显示）
4. 下载识别结果（Excel）

### 标签页 3：Excel 匹配

1. 上传投诉总表（Excel）
2. 选择投诉单号列
3. 点击「开始匹配」
4. 下载匹配结果

## 🔧 常见问题

### Q1: 安装时提示 "未检测到 Python"

**A:** 请先安装 Python 3.7 或更高版本，并确保勾选 "Add Python to PATH"

### Q2: Tesseract 安装后还是提示未安装

**A:** 请检查环境变量是否正确设置，并重启电脑

### Q3: OCR 识别不出来中文

**A:** 确保 Tesseract 安装时勾选了 "Chinese (Simplified)" 语言包

### Q4: 浏览器打不开

**A:** 手动在浏览器中访问 http://localhost:8501

### Q5: 页面卡住不动

**A:** Streamlit 需要重新运行整个脚本，点击按钮会刷新页面，这是正常现象

## 📊 功能说明

### 取证预览
- 上传 .zip 压缩包
- 自动解压
- 预览所有图片
- 勾选需要识别的图片
- 导出选中图片列表

### OCR 识别
- 批量 OCR 识别
- 识别投诉单号
- 识别投诉日期
- 识别反馈产品（需要图片清晰）
- 识别作品名称（需要图片清晰）
- 导出 Excel 结果

### Excel 匹配
- 上传投诉总表
- 按单号匹配
- 填充取证来源（来源压缩包 + 截图名称）
- 填充识别日期
- 导出匹配结果
